# TLOPO-Autologin
Simple python script to login automatically to The Legend of Pirates Online

REQUIRES PYTHON TO BE INSTALLED ON YOUR SYSTEM

Edit the USERNAME, PASSWORD, and GAME_EXE_PATH in the script. Note: You must point to the actual game client executable (tlopo.exe), not the launcher.exe.

How to use this

    Run the script.

    If you don't have 2FA: It launches the game instantly.

    If you DO have 2FA: A text line will appear saying >> Please enter your 2FA Code.

    Open your authenticator app (or check your email), type the code into the window, and hit Enter. The game will then launch.
