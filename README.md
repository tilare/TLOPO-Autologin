# TLOPO-Autologin
Simple python script to login automatically to The Legend of Pirates Online

REQUIRES PYTHON TO BE INSTALLED ON YOUR SYSTEM


How to use this:
1. Edit the USERNAME, PASSWORD, and GAME_EXE_PATH in the script. Note: You must point to the actual game client executable (tlopo.exe), not the launcher.exe.
2. Run the script.
3. If you don't have 2FA: It launches the game instantly.
4. If you DO have 2FA: A text line will appear saying >> Please enter your 2FA Code.
5. Open your authenticator app (or check your email), type the code into the window, and hit Enter. The game will then launch.
